# Snake con React

## Descripción

Implementación del juego clásico Snake utilizando React. El objetivo principal no es solo que el juego funcione, sino que esté bien estructurado usando componentes, props y estado.

## Instrucciones para jugar

1. Abrir el archivo `index.html` en el navegador.
2. Mover la serpiente con el teclado usando:

   * W → arriba
   * A → izquierda
   * S → abajo
   * D → derecha
3. Comer la comida para crecer.
4. Evitar colisiones con las paredes o con la propia serpiente.
5. El juego termina en caso de colisión (game over).
6. El puntaje se muestra durante la partida.
